Selections in D3

With some SVG elements in an SVG canvas we can start to use D3's selection methods to select elements within a page to, as well as other things, change their attributes and styles.

.pumpkin {
	fill: yellow;
	stroke: orange;
	stroke-width: 2;
}
<svg width="200" height="200" style='background: aliceblue;'>
	<g id = "circle_group">
		<circle cx="50" cy="50" r="32" fill="yellow" stroke="orange" stroke-width="4"/>
		<circle cx="110" cy="50" r="16" fill="yellow" stroke="orange" stroke-width="3"/>
		<circle cx="142" cy="50" r="8" class = "pumpkin"/>
		<circle cx="160" cy="50" r="4" fill="yellow" stroke="orange" stroke-width="1"/>
	</g>		
	<g id = "circle_group">
		<rect x="18" y="118" width="64", height = "64" fill="yellow" stroke="orange" stroke-width="4"/>
		<rect id = "goldilocks" x="94" y="134" width="32", height = "32" fill="yellow" stroke="orange" stroke-width="3"/>
		<rect x="134" y="142" width="16", height = "16"  class = "pumpkin" />
		<rect x="156" y="146" width="8", height = "8" fill="yellow" stroke="orange" stroke-width="1"/>
	</g>
</svg>

Try the following commands in the Javascript console to experiment with the select method:

  • d3.select("circle");
  • d3.select("rect");
  • d3.select("circle").style("fill", "blue");
  • d3.select("rect").style("fill", "red");
  • d3.select(".pumpkin").attr("cy", 30);
  • d3.select("#goldilocks").attr("width", 20);
  • d3.select("#goldilocks").attr("width", 40).attr("height", 50).attr("fill", "green");
  • d3.select(".pumpkin").style("stroke", "black");
  • d3.select("circle").select("circle").style("fill", "blue");
  • d3.select("#square_group").select("rect").style("fill", "purple");

Try the following commands in the Javascript console to experiment with the selectAll method:

  • d3.selectAll("circle");
  • d3.selectAll("rect").style("fill", "red");
  • d3.selectAll("circle").attr("r", 10);
  • d3.selectAll(".pumpkin").style("stroke", "green").style("stroke-width", 6);
  • d3.selectAll("circle").attr("r", 10);

Selections don't have to just be SVG elements, they can be any element on the page:

  • d3.select("h1");
  • d3.select("h1").text("All Your Base Are Belong To Us");
  • d3.selectAll("p").style("font-family", "cursive");
