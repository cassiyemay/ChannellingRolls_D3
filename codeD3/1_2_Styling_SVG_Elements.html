Styling SVG Elements

Fills and Strokes

The appearance of SVG shapes can be controlled using style propeties. Style properties can be specified directly in the shape elements themselves.

<svg width="200" height="100" style='background: aliceblue;'>
	<circle cx="50" cy="50" r="32" fill="yellow" stroke="orange" stroke-width="5"/>
</svg>
<svg width="200" height="100" style='background: aliceblue;'>
	<rect x="0" y="0" width="30" height="30" fill="purple" stroke = "grey"/>
	<rect x="20" y="5" width="30" height="30" fill="blue" stroke = "green"/>
	<rect x="40" y="10" width="30" height="30" fill="green" stroke = "orange"/>
	<rect x="60" y="15" width="30" height="30" fill="yellow" stroke = "black"/>
	<rect x="80" y="20" width="30" height="30" fill="red" stroke = "purple"/>
</svg>

Or defined styles including style properties can be applied to shape elements.

.pumpkin {
	fill: yellow;
	stroke: orange;
	stroke-width: 5;
}	
<svg width="200" height="100" style='background: aliceblue;'>
	<circle cx="50" cy="50" r="32" class="pumpkin"/>
</svg>

It can be useful to controlling transparency. Opacity can be specified either through the opacity property, or by using an alpha channel in colour defintions.

<svg width="200" height="100" style='background: aliceblue;'>
	<circle cx="25" cy="25" r="20" fill="rgb(128, 0, 128)" opacity = "1.0"/>
	<circle cx="50" cy="25" r="20" fill="rgb(0, 0, 255)" opacity = "0.75"/>
	<circle cx="75" cy="25" r="20" fill="rgb(0, 255, 0)" opacity = "0.5"/>
	<circle cx="100" cy="25" r="20" fill="rgb(255, 255, 0)" opacity = "0.25"/>
	<circle cx="125" cy="25" r="20" fill="rgb(255, 0, 0)" opacity = "0.1"/>
</svg>
<svg width="200" height="100" style='background: aliceblue;'>
	<circle cx="25" cy="25" r="20" fill="rgba(128, 0, 128, 1.0)"/>
	<circle cx="50" cy="25" r="20" fill="rgba(0, 0, 255, 0.75)"/>
	<circle cx="75" cy="25" r="20" fill="rgba(0, 255, 0, 0.5)"/>
	<circle cx="100" cy="25" r="20" fill="rgba(255, 255, 0, 0.25)"/>
	<circle cx="125" cy="25" r="20" fill="rgba(255, 0, 0, 0.1)"/>
</svg>

Opacity can be set individually for fill and stroke.

<svg width="200" height="100" style='background: aliceblue;'>
	<circle cx="25" cy="25" r="20" fill="rgba(128, 0, 128, 0.75)" stroke="rgba(0, 255, 0, 0.25)" stroke-width="10"/>
	<circle cx="75" cy="25" r="20" fill="rgba(0, 255, 0, 0.75)" stroke="rgba(0, 0, 255, 0.25)" stroke-width="10"/>
	<circle cx="125" cy="25" r="20" fill="rgba(255, 255, 0, 0.75)" stroke="rgba(255, 0, 0, 0.25)" stroke-width="10"/>
</svg>
<svg width="200" height="100" style='background: aliceblue;'>
	<circle cx="25" cy="25" r="20" fill="purple" stroke="green" stroke-width="10" opacity="0.9"/>
	<circle cx="65" cy="25" r="20" fill="green" stroke="blue" stroke-width="10" opacity="0.5"/>
	<circle cx="105" cy="25" r="20" fill="yellow" stroke="red" stroke-width="10" opacity="0.1"/>
</svg>
<svg width="200" height="100" style='background: aliceblue;'>
	<circle cx="25" cy="25" r="20" fill="rgba(128, 0, 128, 0.75)" stroke="rgba(0, 255, 0, 0.25)" stroke-width="10"/>
	<circle cx="65" cy="25" r="20" fill="rgba(128, 0, 128, 0.75)" stroke="rgba(0, 255, 0, 0.25)" stroke-width="10" opacity="0.5"/>
	<circle cx="105" cy="25" r="20" fill="rgba(128, 0, 128, 0.75)" stroke="rgba(0, 255, 0, 0.25)" stroke-width="10" opacity="0.2"/>
</svg>

Styling Lines

We have a lot of control over the appearance of lines

<svg width="200" height="120" style='background: aliceblue;'>
	<line x1="20" y1="20" x2 = "180" y2 = "20" stroke-width="2" stroke = "black"/>
	<line x1="20" y1="30" x2 = "180" y2 = "30" stroke-width="5" stroke = "black"/>
	<line x1="20" y1="50" x2 = "180" y2 = "50" stroke-width="10" stroke = "black"/>
	<line x1="20" y1="80" x2 = "180" y2 = "80" stroke-width="20" stroke = "black"/>
</svg>
<svg width="200" height="120" style='background: aliceblue;'>
	<line x1="20" y1="20" x2 = "180" y2 = "20" stroke-linecap="butt" stroke = "black" stroke-width="10"/>
	<line x1="20" y1="50" x2 = "180" y2 = "50" stroke-linecap="round" stroke = "black" stroke-width="10"/>
	<line x1="20" y1="80" x2 = "180" y2 = "80" stroke-linecap="square" stroke = "black" stroke-width="10"/>
</svg>
<svg width="200" height="120" style='background: aliceblue;'>
	<line x1="20" y1="20" x2 = "180" y2 = "20" stroke-dasharray="1 1" stroke = "black" stroke-width="5"/>
	<line x1="20" y1="40" x2 = "180" y2 = "40" stroke-dasharray="10 4 4 4" stroke = "black" stroke-width="5"/>
	<line x1="20" y1="60" x2 = "180" y2 = "60" stroke-dasharray="10 10" stroke = "black" stroke-width="5"/>
	<line x1="20" y1="80" x2 = "180" y2 = "80" stroke-dasharray="20 10 5 5 5 10" stroke = "black" stroke-width="5"/>
	<line x1="20" y1="100" x2 = "180" y2 = "100" stroke-dasharray="5 5 5 5 5 5 25 5 25 5 25 5 5 5 5 5 5 20" stroke = "black" stroke-width="5"/>
</svg>

Styling Text

The text element gives us a good deal of control over text elements

<svg width="200" height="140" style='background: aliceblue;'>
	<text x="20" y="20" font-family = "serif">D3 is fun</text>
	<text x="20" y="40" font-family = "sans-serif">D3 is fun</text>
	<text x="20" y="60" font-family = "Times  New Roman">D3 is fun</text>
	<text x="20" y="80" font-family = "Impact">D3 is fun</text>
	<text x="20" y="100" font-family = "Georgia">D3 is fun</text>
	<text x="20" y="120" font-family = "cursive">D3 is fun</text>
</svg>
<svg width="200" height="120" style='background: aliceblue;'>
	<text x="20" y="20" font-size = "6">D3 is fun</text>
	<text x="20" y="40" font-size = "12">D3 is fun</text>
	<text x="20" y="60" font-size = "24">D3 is fun</text>
	<text x="20" y="100" font-size = "48">D3 is fun</text>
</svg>
<svg width="200" height="120" style='background: aliceblue;'>
	<text x="20" y="20" fill = "orange" stroke = "red" opacity = "1.0" font-size = "24" font-family = "Arial Black">D3 is fun</text>
	<text x="30" y="40" fill = "yellow" stroke = "blue" opacity = "0.75" font-size = "24" font-family = "Arial Black">D3 is fun</text>
	<text x="40" y="60" fill = "green" stroke = "blue" opacity = "0.5" font-size = "24" font-family = "Arial Black">D3 is fun</text>
	<text x="50" y="80" fill = "yellow" stroke = "red" opacity = "0.25" font-size = "24" font-family = "Arial Black">D3 is fun</text>
	<text x="60" y="100" fill = "blue" stroke = "red" opacity = "0.1" font-size = "24" font-family = "Arial Black">D3 is fun</text>
</svg>
<svg width="200" height="120" style='background: aliceblue;'>
	<text x="20" y="60" font-size = "24" font-family = "Arial Black" opacity = "0.6">D3 is fun</text>
	<text x="20" y="40" transform="rotate(30 20,40)" fill = "red" opacity = "0.6" font-size = "24" font-family = "Arial Black">D3 is fun</text>
	<text x="60" y="100" transform="rotate(-50 60,100)" fill = "blue" opacity = "0.6" font-size = "24" font-family = "Arial Black">D3 is fun</text>
</svg>
