Drawing SVG Elements

Drawing on an SVG Canvas

We will do all of our drawing on an SVG canvas element. Canvas elements are easily created in a HTML page as SVG elements.

<svg width="500" height="50" style='background: aliceblue;'>

</svg>

Once we have a canvas we can draw basic shapes onto it by adding a shape element within the SVG.

<svg width="200" height="200" style='background: aliceblue;'>
	<circle cx="110" cy="125" r="40"/>
</svg>

Watch out that shapes can disappear off the edge of a canvas.

<svg width="200" height="200" style='background: aliceblue;'>
	<circle cx="110" cy="125" r="40"/>
</svg>

SVG Basic Shapes

SVG supports a set of basic shapes with which we can draw onto an SVG canvas.

Rectangle

<svg width="200" height="200" style='background: aliceblue;'>
	<rect x="30" y="50" width="80" height="100"/>
</svg>

Rounded Rectangle

<svg width="200" height="200" style='background: aliceblue;'>
	<rect x="30" y="50" width="80" height="100" rx = "10", ry = "10"/>
</svg>

Circle

<svg width="200" height="200" style='background: aliceblue;'>
	<circle cx="110" cy="125" r="40"/>
</svg>

Ellipse

<svg width="200" height="200" style='background: aliceblue;'>
	<ellipse cx="110" cy="75" rx="80" ry="25"/>
</svg>

Line

Lines must have a stroke to be visible.

<svg width="200" height="200" style='background: aliceblue;'>
	<line x1="10" y1="10" x2="190" y2="190" stroke = "black"/>
</svg>

Polyline

<svg width="200" height="200" style='background: aliceblue;'>
	<polyline points = "30,10, 160,75, 172,130, 80,185, 20,90" stroke = "black" fill=none/>
</svg>

Polygon

<svg width="200" height="200" style='background: aliceblue;'>
	<polygon points = "30,10, 160,75, 172,130, 80,185, 20,90"/>
</svg>

Path

<svg width="200" height="200" style='background: aliceblue;'>
	<path d="M 10 80 C 40 10, 65 10, 95 80 S 150 150, 180 80 L 180 190 L 10 190 Z"/>
</svg>

Text

<svg width="200" height="200" style='background: aliceblue;'>
	<text x="250" y="25">SVG Text!!!</text>
</svg>
