SVG Groups

SVG elements can be combined into groups, which allows things like styling to applied in one go.

<svg width="300" height="100" style='background: aliceblue;'>
	<g>
		<circle cx="210" cy="90" r="40"/>
		<rect x="200" y="120" width="20", height = "80"/>
		<rect x="20" y="100" width="120", height = "100"/>
		<polygon points="20,98, 80,50, 140,98"/>
	</g>
</svg>
